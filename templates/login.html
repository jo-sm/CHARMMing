About CHARMMing

CHARMMing, which stands for CHARMM interface and graphics, is an intergrated set of tools that performs chemical structure calculations and analysis. It incorpoates freely available tools such as JSmol and GENREF, an automatic residue topology file generator that generates information not supported with CHARMM force fields.
